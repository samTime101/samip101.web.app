Contribute to User Gate 101

This project is an interactive login system that allows users to create an account, log in, and access their personalized information. It provides a secure and efficient way to manage user authentication.

We invite you to contribute to this project and help make it even better! Whether you have ideas for new features, suggestions for improvement, or want to report any issues you encounter, your contributions are valuable.

To contribute, please visit our GitHub repository and follow the guidelines for submitting pull requests. We appreciate your involvement in making this project a success!

Goals:

  • Password Hashing: Passwords are securely hashed and stored in the file.
  • Password Complexity: Passwords must meet complexity requirements (length, uppercase, lowercase, numbers, special characters).
  • Input Validation: User input is validated to ensure it meets the required format and length for usernames and passwords.
  • Account Locking: After multiple failed login attempts, accounts are locked for a specified duration.
  • User Profile: Users can provide additional information (email, profile picture) stored in their account.
  • Account Deletion: Users can delete their account, removing their information from the file.
  • Password Reset: Users can reset their password through a secure email reset link.
  • Secure File Handling: File permissions and encryption techniques are used to secure the account file.
  • Role-Based Access Control: Different access privileges are assigned based on user roles.
  • Session Timeout: Users are automatically logged out after a period of inactivity.
